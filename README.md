Учебный проект по курсу "Финансовый учёт и отчётность" Факультета экономических наук НИУ ВШЭ по финансовому анализу компании "Норникель", создан совместно в результате групповой работы.
В файле .xlsx содержится собранная финансовая отчётность "Норникеля" и её конкурентов и в России, и во всём мире и провели их анализ. В файле .pdf - презентация, содержащая основные выводы работы.
